Icons

Font Awesome

Accessibility and font awesome icons: https://fontawesome.com/how-to-use/on-the-web/other-topics/accessibility

If your icons are purely decorative, you’ll need to manually add an aria-hidden attribute to each of your icons so they’re accessible.
        
            <i class="fas fa-camera-retro" aria-hidden></i>
If your icons have semantic meaning, you’ll need to manually add a few things so that your icon is appropriately accessible: aria-hidden attribute + Provide a text alternative inside a span (or similar) element. Also include appropriate CSS to visually hide the element while keeping it accessible to assisitive technologies + title attribute on the icon to provide a tooltip for sighted mouse users. 
        
            <i aria-hidden class="fas fa-car" title="Time to destination by car"></i>
            <span class="sr-only">Time to destination by car:</span>
            <span>4 minutes</span>
